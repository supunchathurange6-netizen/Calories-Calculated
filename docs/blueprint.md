# **App Name**: Ceylanta Calories

## Core Features:

- User Profile Setup: Collects user data (name, age, gender, height, weight, activity level, goal) to personalize calorie calculations.
- Daily Calorie Calculator: Automatically calculates daily calorie needs, protein/carbs/fat suggestions, and displays remaining/consumed/burned calories based on user profile and food logging.
- Food Logging: Allows users to search and select Sri Lankan and international foods, specify portion sizes, and add them to breakfast/lunch/dinner logs.
- Manual Food Entry: Enables users to manually enter food details (name, calories) for foods not found in the database, saving them for future use.
- Personalized Food Suggestions: Suggests food based on user preferences and dietary goals, incorporating a tool that recommends Sri Lankan dishes and international options for variety.
- Daily Summary Screen: Provides a visual summary of calories eaten, calories remaining, and progress with a progress bar and visual alerts for exceeding limits.
- Progress Tracking: Tracks weight and average daily calorie intake weekly/monthly with simple charts.

## Style Guidelines:

- Primary color: Vibrant orange (#FF7043) to evoke a sense of energy and appetite.
- Background color: Light peach (#FFE7D9), a soft and desaturated tint of orange that provides a welcoming backdrop.
- Accent color: Analogous yellow-orange (#FFAB40), which creates visual interest without overwhelming the design.
- Font pairing: 'Poppins' (sans-serif) for headlines, and 'PT Sans' (sans-serif) for body text.
- Use food-related icons for food categories and meal types. The icons should be simple and easily recognizable.
- Use a clean and intuitive layout with clear sections for profile, daily summary, food logging, and progress tracking.
- Subtle transitions when adding food or updating progress to provide visual feedback without being distracting.
